========================================================================
    DYNAMIC LINK LIBRARY : DSWrapper Project Overview
========================================================================

Systemove poziadavky:

	* Windows 7 (koli Microsoft MPEG-2 Encoder)

	* .NET Framework 4.0

Volitelne:

	* Pre integraciu titulkov a zmenu rozlisenia videa FFDSHOW
	  http://sourceforge.net/projects/ffdshow-tryout/

	* Pre podporu WebM formatu - WebM filter a Ogg Vorbis filter
	  http://www.webmproject.org/tools/#directshow_filters
	  http://xiph.org/dshow/

	* Pre lepsiu podporu vstupnych suborov sa odporuca
	  codec pack, napr.: 
	  http://www.codecguide.com/download_kl.htm

Upozornenie:

	* Na klientskom PC treba prilozit aj Microsoft C++ Runtime Library súbory 
	  "msvcp100.dll" a "msvcr100.dll" nachadzajuce sa v adresary DSWrapperApp
